- 설정 activity에 어떠한 항목?
  사용자 정보 수정(챗봇이 불러주는 이름..), 챗봇 선택, 데이터 초기화, 로그아웃
    
- 사용자 계정을 제공?
    - 제공한다면 어디까지 저장?
      계정 정보, 선택한 챗봇, 대화 내역, 사용자 스케쥴 --> DB 저장

- 사용자로부터 어떠한 데이터 입력?
    로그인(네이버, 페이스북, 카카오톡), 닉네임 입력

- 누가 어떠한 activity 담당?
    챗봇 선택 -> 수현 [ ~ 다음 주 월 ]
    로그인 기능 구현(카카오톡, 구글, 페이스북) -> 이유리 [금 ~ 일]
    채팅창 -> 한유리 [ 수 ~ 금 ] 
    
- 전체 앱 디자인 -> 수진 [ ~ 일]

- 챗봇 바꿀 경우 이전 대화 내역은 어떻게 처리?
    한유리가 고민하기
   
- auto layout (깨지지 않도록)
    이유리가 찾아보기
